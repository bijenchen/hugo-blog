# DNS 是什麼 ? 他們是如何運作 ?

## DNS 是什麼
Domain Name System ，紀錄數字IP與文字網址對應關係的地方。
概念上像是把我家的經緯度座標變成中文地址的地政系統一樣的東西。

## 如何運作
就是把「 IP 」與「對應的網址」的對應關係存起來，讓人類可以雙向查詢對應內容。
